# llama3-on-apple-silicon

This is a sample project for generating text with Meta's Llama3 on Apple Silicon.

## Requirements

- Rye : [astral-sh/rye: a Hassle-Free Python Experience](https://github.com/astral-sh/rye)
- Python 3.12

## Installation

```bash
rye sync
```

## Usage

```bash
rye run python ./src/llama3_on_apple_silicon/main.py
```
